# Accessible Quiz Builder

An accessible quiz builder application built with React, TypeScript, and Vite, focused on creating inclusive and WCAG-compliant quiz experiences.

## Prerequisites

- Node.js (v20 or higher recommended)
- npm, yarn, or pnpm

## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Building

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Testing

### Unit Tests

Run unit tests with Jest:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```

### End-to-End Tests

Run E2E tests with Playwright:

```bash
npm run test:e2e
```

## Code Quality

### Linting

Check for linting issues:

```bash
npm run lint
```

Automatically fix linting issues:

```bash
npm run lint:fix
```

### Formatting

Format code with Prettier:

```bash
npm run format
```

### Pre-commit Hooks

This project uses Husky and lint-staged to automatically lint and format code before commits. The hooks will run automatically when you commit changes.
